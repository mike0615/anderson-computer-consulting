## Overview

Security is not an afterthought at ACC — it is the foundation of every design. We bring government-grade security practices to enterprise environments, implementing comprehensive monitoring, hardening, and compliance frameworks.

## Core Capabilities

| Capability | Technologies |
|---|---|
| SIEM & Log Management | Graylog, Wazuh |
| Endpoint & Host Security | Wazuh agents, auditd, SELinux |
| Identity & Access | FreeIPA, LDAP, MFA |
| Compliance Frameworks | ICD 503, FISMA, NIST 800-53 |
| Network Security | Juniper SRX, zone-based firewalls, IDS |
| Vulnerability Management | OpenVAS, custom scanning scripts |

- Security architecture design and review
- ICD 503 / FISMA compliance assessment and implementation
- SIEM deployment and tuning (Graylog + Wazuh)
- Security hardening for Rocky Linux, network devices, and hypervisors
- Incident response planning and tabletop exercises
- Air-gapped security monitoring environments

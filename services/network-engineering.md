## Overview

Anderson Computer Consulting delivers enterprise-grade network engineering for organizations that need complex, reliable, and secure network infrastructure. We design, deploy, and operate multi-site networks with a security-first, air-gapped-capable approach.

## Core Capabilities

| Capability | Technologies |
|---|---|
| Routing & Switching | Cisco Catalyst 9300/8300, Juniper SRX 300/340/380 |
| Network Security | ACLs, Zone-Based Firewalls, VPN, IDS/IPS |
| WAN Architecture | Multi-site MPLS, SD-WAN concepts, redundant WAN |
| Network Monitoring | LibreNMS, Zabbix, Graylog, Wazuh |
| Automation | Ansible network modules, GitLab CI/CD |
| Documentation | NetBox (IPAM/DCIM), network diagrams |

## What We Deliver

- **Network architecture design** — logical and physical topology, IP addressing, VLAN design
- **Multi-site enterprise builds** — centralized and distributed service models
- **HA and redundancy** — HSRP, VRRP, LACP, redundant uplinks
- **Air-gapped network design** — fully offline-capable environments for government/classified use
- **Network automation** — Ansible playbooks for config management and compliance
- **Ongoing operations support** — monitoring, alerting, troubleshooting

## Common Problem Areas

We regularly solve:

- ARP/routing asymmetry in complex multi-subnet environments
- NAT traversal in multi-site architectures
- Juniper SRX policy and zone configurations
- Cisco IOS/IOS-XE BGP, OSPF, and EIGRP deployments
- WAN emulation and packet capture for testing and validation

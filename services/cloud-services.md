## Overview

Not every organization can or should use public cloud. ACC specializes in private cloud and air-gapped cloud alternatives that give you the agility of cloud without the internet dependency or compliance risk.

## Solutions

| Solution | Description |
|---|---|
| Private Cloud | XCP-ng + CloudStack — self-hosted, enterprise-grade |
| Container Platform | Docker + private registry — fully air-gapped |
| Artifact Management | Offline Rocky Linux repo mirrors, Docker registry |
| Hybrid Architecture | On-prem primary + controlled cloud egress |
| Infrastructure as Code | Ansible + Terraform for reproducible builds |

## Why Private Cloud with ACC

- **Air-gap capable** — full offline operation, no internet required
- **Open source** — no vendor lock-in, no license surprises
- **Government ready** — meets ICD 503 and FISMA requirements
- **Automation first** — GitLab CI/CD, Ansible, reproducible builds

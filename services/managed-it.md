## Overview

ACC provides fully managed IT services for organizations that need enterprise-grade infrastructure management without building a full internal team. We monitor, maintain, and operate your infrastructure so you can focus on your mission.

## What's Included

| Service | Details |
|---|---|
| 24/7 Monitoring | LibreNMS, Zabbix, Wazuh — proactive alerting |
| Patch Management | Automated patching for Rocky Linux, network devices |
| Backup & Recovery | Tested backup procedures, documented recovery runbooks |
| Help Desk Support | Tier 1-3 support for infrastructure issues |
| Change Management | Git-tracked configuration changes, rollback capability |
| Documentation | NetBox IPAM, Wiki.js runbooks, network diagrams |

## Technology Stack

We manage environments built on:

- **Servers:** Rocky Linux 9.x on bare metal, KVM, and XCP-ng
- **Networking:** Cisco and Juniper multi-site environments
- **Identity:** FreeIPA with LDAP integration
- **Monitoring:** LibreNMS + Zabbix + Graylog + Wazuh
- **Automation:** Ansible + GitLab CI/CD

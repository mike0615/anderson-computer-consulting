## Our Technology Stack

ACC builds solutions on proven, open-source, enterprise-grade technology:

| Category | Technology | Role |
|---|---|---|
| Hypervisor | XCP-ng | Primary virtualization platform |
| OS | Rocky Linux | Server operating system |
| Identity | FreeIPA | LDAP, DNS, PKI, Kerberos |
| Networking | Cisco / Juniper | Switching, routing, firewalls |
| Monitoring | LibreNMS + Zabbix | Network and infrastructure monitoring |
| SIEM | Graylog + Wazuh | Log management and security monitoring |
| Automation | Ansible + GitLab | Configuration management and CI/CD |
| Collaboration | Mattermost + Openfire | Team messaging and XMPP |
| Documentation | NetBox + Wiki.js | IPAM/DCIM and knowledge management |
| Remote Access | Apache Guacamole | Browser-based remote desktop |
